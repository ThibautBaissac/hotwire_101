## 2. QUICK START
1. Clone the repository
```bash
git clone git@github.com:ThibautBaissac/hotwire_101.git
```
2. Run bundle install
```bash
bundle
```

3. Run yarn
```bash
yarn install
```

4. Run rails db commands
```bash
rails db:drop db:create db:migrate db:seed
```
